# Marketing Site — UI Kit

A Phantom-style marketing landing page built from the system's primitives and tokens.

**Sections:** sticky blurred nav with pill links → hero with purple radial glow + floating mark → 3-up feature cards → gradient CTA panel → footer.

## Files
- `index.html` — assembles the page
- `sections.jsx` — `Nav`, `Hero`, `Features`, `CTA`, `Footer`

## Notes
- Dark theme, 1200px container, 24px gutters.
- Copy follows the brand voice: plain, warm, active verbs ("Send money, simply").
- Wordmark/mark are placeholders — swap in real brand assets.
