Text field with optional label, helper text, and error state. Focus adds a brand glow halo.

```jsx
<Input label="Amount" placeholder="0.00" helperText="Available: 12.4 SOL" />
<Input label="Recipient" error helperText="Invalid address" />
```

Props: `label`, `helperText`, `error`, `leadingIcon`, `trailingIcon`, plus all native input attributes. 16px radius, surface background.
